\# FlyingSBMap



Reverse engineering tools for the FlyingSB map format.



Current features:



\- Inspect map records

\- Render maps

\- Compare maps

\- Search by record signature

\- Edit individual fields

\- Render edited maps

\- Analyse record statistics



Goal:



Fully decode the proprietary map format so hidden routes, transparent objects, collision data, triggers and other hidden map information can be visualised.
